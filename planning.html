Planeación

Objetivo General

Conocer las principales etiquetas HTML 5 para orfanizar de manera semantica el contenido de una página web.

Objetivos específicos

  • Manejar los contenidos generales con la etiqueta SECTION.
  • Ubicar correctamente el contenido para la cabecera y el pie de página, etiquetas HEADER y FOOTER respectivamente.
  • Establecer el menú dentro de la etiqueta ASIDE.
  • Identificar el contenido principal para ubicarlo dentro de la etiqueta ARTICLE.

Contenido

Con la llegada de HTML5 y con el auge de la web semántica se establecieron nuevas etiquetas que dieran mejor significado a la estructura básica de una web, recomendando dejar de usar la etiqueta <div> en muchos de los casos.

Estas son algunas de las nuevas etiquetas:

  • <header>...</header> Es el encabezado de la pagina. Contiene los elementos mas importantes de identidad y navegación de la página.
  • <footer>...</footer> Es el pie de página. Contiene normalmente la información de contacto, el copyright u otros enlaces de interés.
  • <section>...</section> Nos permite definir diferentes secciones dentro de la misma página o pequeños bloques de contenido.
  • <aside>...</aside> Representa una parte de la página que contiene información que no está relacionada con el contenido principal. Por ejemplo publicidad, enlaces externos, calendarios incluso se puede ubicar el menú.
  • <article>...;</article> En un blog puede representar el texto de cada artículo, en un foro cada mensaje publicado por un usuario o en un periódico puede ser cada noticia.
